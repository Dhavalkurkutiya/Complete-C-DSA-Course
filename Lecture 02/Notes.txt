1. What is cpp ?
Ans: C++ is a general-purpose programming language created by Bjarne Stroustrup as an extension of the C programming language, or "C with Classes". The language has expanded significantly over time, and modern C++ now has object-oriented, generic, and functional features in addition to facilities for low-level memory manipulation.

2. What is the difference between C and C++ ?
Ans: C is a procedural programming language, whereas C++ is an object-oriented programming language. C is a subset of C++. C++ is a superset of C. C follows top-down programming approach. C++ follows bottom-up programming approach.

3. What is variable ?
Ans: A variable is a storage location (identified by a memory address) paired with an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.

4. What is Data Type ?
Ans: A data type is a classification of data which tells the compiler or interpreter how the programmer intends to use the data. Most programming languages support various types of data, including integer, real, character or Boolean.

5. What is Operator ?
Ans: An operator in a programming language is a symbol that tells the compiler or interpreter to perform specific mathematical, relational or logical operation and produce final result.
